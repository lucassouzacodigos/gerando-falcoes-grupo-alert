Este documento descreve como nosso grupo utilizará IA (como GitHub Copilot) durante o projeto.

## Princípios
- Usar IA para acelerar ideias e documentação.
- Revisar todo conteúdo gerado pela IA.
- Não usar IA para substituir nosso aprendizado ou autoria.
- Garantir transparência no uso de IA.

## O que iremos usar
- Sugestões de texto.
- Geração de estrutura inicial de código.
- Explicações técnicas.

## O que não iremos usar
- Geração automática sem revisão humana.
- Conteúdo sensível ou que viole privacidade.
